  • Last Emoji Reaction History (stored for a maximum of 2 hours or less)

Why Do We Need the Data and How Is It Used?

When a command is invoked, we store the message content for a maximum of 14 days for debugging purposes. We also store a maximum of 18 entries for edited messages and sniping messages that will expire in two hours or less in volatile memory.

Guild IDs, Channel IDs, Role IDs, User IDs, and Message IDs are stored for our system to aggregate values to find data.

Nickname, Username, and Avatar changes are logged for the "namehistory" and "avatarhistory" commands to function. Users can clear this data themselves at any time.

Guild name changes are logged for the "gnames" command to function. Server administrators can clear this data themselves at any time.
